## 💡 Pro Tips for Your Report

### Diagram Captions
Always add captions below diagrams:
- "Figure 1: Use Case Diagram showing system actors and their capabilities"
- "Figure 2: Gantt Chart depicting the 110-day development timeline"
- "Figure 3: System Architecture illustrating the three-tier design"
- "Figure 4: Class Diagram showing database models and relationships"
- "Figure 5: Sequence Diagram for user authentication flow"
- "Figure 6: Activity Diagram for quiz taking workflow"

### Diagram References
Reference diagrams in your text:
- "As shown in Figure 1, the system supports three types of users..."
- "The development timeline (Figure 2) spans 110 days across seven phases..."
- "Figure 3 illustrates the three-tier architecture employed..."

### Image Quality
For best results:
- Use high resolution screenshots
- Ensure good contrast
- Make diagrams readable at print size
- Consider landscape orientation for wider diagrams
