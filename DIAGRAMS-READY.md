## 📸 How to Use in Your Report

### Step 1: View on GitHub
1. Go to your repository on GitHub
2. Navigate to `docs/diagrams/`
3. Click on any diagram file

### Step 2: Diagrams Render Automatically
- GitHub automatically renders Mermaid diagrams
- No additional tools needed
- Professional, clean appearance

### Step 3: Screenshot for Report
1. Take screenshot of the rendered diagram
2. Save as PNG or JPG
3. Insert into your report document
4. Add caption (e.g., "Figure 1: Use Case Diagram")

### Alternative: Use Mermaid Live Editor
1. Copy the Mermaid code from the file
2. Go to https://mermaid.live/
3. Paste the code
4. Download as PNG or SVG
5. Insert high-quality image into report
